About LinuxOnBrowser

This is a virtual machine running GNU/Linux
based on javascript with the forked v86.js.

Feature
  • Using TinyCore with GUI and Kernel 5.x
  • Network Supported
  • Support saving/loading state
  • Beautiful UI
  • 100% OpenSource & Free
